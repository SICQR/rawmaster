# RAWMASTER Demo
**Smash Daddys Audio Tools č Strip it back. Own the stems.**

Demo version. Tracks are cropped to 90 seconds.

**[Full version č no limits č at scanme2.gumroad.com](https://scanme2.gumroad.com)** č Ż19 one-time.

## What it does

- Remaster your track (spectral gating + LUFS normalisation + hard limiter)
- Separate into 4 or 6 stems (HTDemucs fine-tuned)
- Extract MIDI from bass (and optionally vocals) via Basic-Pitch
- Detect BPM and musical key

Everything runs on the server. No account needed.
